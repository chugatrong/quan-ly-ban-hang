File Excel Mẫu

Vì giới hạn của GitHub Pages, chúng tôi không thể cung cấp trực tiếp file Excel để tải xuống. Tuy nhiên, bạn có thể dễ dàng tạo file Excel mẫu theo hướng dẫn dưới đây.

Cách tạo file Excel mẫu

  1. Mở Microsoft Excel hoặc Google Sheets
  2. Tại ô A1, nhập Tên món
  3. Tại ô B1, nhập Giá
  4. Nhập danh sách món ăn của bạn, ví dụ:
    • A2: Phở bò, B2: 50000
    • A3: Bún chả, B3: 45000
    • A4: Cơm rang, B4: 40000
    • A5: Gà rán, B5: 60000
    • A6: Bánh mì, B6: 25000
    • A7: Cà phê, B7: 30000
  5. Lưu file với định dạng .xlsx

Lưu ý quan trọng:

  • Giá nên được nhập dưới dạng số, không có đơn vị tiền tệ hoặc dấu phân cách hàng nghìn.
  • Các tên cột có thể thay đổi linh hoạt: "Tên món"/"Tên"/"Món" và "Giá"/"Đơn giá"/"Giá tiền".
  • Bạn có thể thêm bao nhiêu món tùy thích, hệ thống sẽ đọc tất cả.

Nếu bạn muốn có sẵn mẫu, bạn có thể tạo file Excel mới và sao chép các giá trị từ bảng trên.
